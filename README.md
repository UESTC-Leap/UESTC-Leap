### 投稿/修改

#### 方式一：通过Github

- 注：您的开发环境须有npm
- Fork本仓库
- 在仓库目录下安装依赖: `npm install` 或 `yarn install`
- 本地运行: `npm run serve` 或 `yarn serve`
- 创建分支: `git checkout -b my-new-feature`
- 提交改动: `git commit -am 'Add some feature'`
- 上传分支: `git push origin my-new-feature`
- 提交Pull Request
